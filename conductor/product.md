# Product Definition: Under Sports (Bravio Ecommerce)

## Visão Geral
O projeto é um sistema de E-commerce robusto focado em artigos esportivos, com integração completa para pagamentos via PIX e uma experiência de usuário moderna.

## Objetivos Principais
- Fornecer uma plataforma de vendas rápida e responsiva.
- Integração fluída com gateways de pagamento (SafefyPay, MercadoPago).
- Sistema de e-mails automatizados para confirmação de pedidos e pagamentos.
- Deploy automatizado e resiliente via Dokploy.

## Personas
- **Clientes**: Buscam produtos esportivos com facilidade de pagamento.
- **Administradores**: Gerenciam pedidos e garantem o fluxo de entrega.

## Funcionalidades Chave
- Carrinho de compras.
- Checkout especializado para PIX.
- Notificações de pagamento.
- Catálogo de produtos categorizado.
